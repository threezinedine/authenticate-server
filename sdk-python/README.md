# ntt-client-auth

A high-performance, asynchronous FastAPI Client SDK for statelessly verifying JWT Access Tokens generated by the Centralized Authentication Service.

## Features

- **Zero-Network Authentication**: Validates tokens mathematically without querying the central database.
- **Asynchronous JWKS Caching**: Automatically fetches and caches the public `jwks.json` to handle Identity Server key rotations seamlessly.
- **FastAPI Native**: Provides plug-and-play `Depends()` functions to inject strongly-typed User properties directly into your routes.
